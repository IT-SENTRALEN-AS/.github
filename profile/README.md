### Om oss

**IT-SENTRALEN AS** utvikler skreddersydde IT-løsninger for norske bedrifter. Vi spesialiserer oss på kassesystemer, nettbutikkintegrasjoner, SaaS-plattformer og forretningssoftware.

Vi bygger med moderne teknologi — TypeScript, React, Next.js, Supabase, Medusa og Stripe — og leverer alt fra domeneadministrasjonsverktøy til SAF-T-kompatible kassesystemer.

### Hva vi gjør

| Område | Beskrivelse |
|:-------|:------------|
| **Kassesystemer (POS)** | SAF-T-kompatible kasseløsninger med digitale kvitteringssignaturer |
| **Netthandel** | WooCommerce- og Medusa-integrasjoner med sanntidssynkronisering mot POS |
| **SaaS-plattformer** | Flerbrukersystemer med Stripe-betaling og Supabase-backend |
| **API-integrasjoner** | REST- og SOAP-integrasjoner for norske forretningssystemer (Tripletex, Norid, Domeneshop) |
| **Domeneadministrasjon** | Registrarverktøy med Norid EPP- og Domeneshop API-integrasjon |
